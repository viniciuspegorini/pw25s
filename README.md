# Aulas da disciplina de Programação para Web - PW25S-5SI

## Back-end 

API REST desenvolvida utilizando o *framework* **Spring**.

### Softwares
	- JDK 11 ou superior
	- IDE:
		- ItelliJ IDEA
		- Spring Tools Suite 4
		- Eclipe for JavaEE ...
	- SDBG:
		- Postgresql
	- Ferramenta para testar a API:
		- Postman
		- Insomnia
	- Git
	- Docker
